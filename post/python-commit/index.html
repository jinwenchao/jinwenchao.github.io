vim 添加头文件注释

编辑C:\Program Files\Vim\_vimrc文件,在文件末未追加如下内容：

function HeaderPython()
call setline(1, "#!/usr/bin/env python")
call append(1, "#-*- coding:utf8 -*-")
call append(2, "#")
call append(3, "# Copyright	: Copyright 2021 Inc.")
call append(4, "# File			: .py")
call append(5, "# Introduce	: ")
call append(6, "#")
call append(7, "# Version		: V1.0.1")
call append(8, "# Author		: shalter")
call append(9, "# Time		: ".strftime('%Y-%m-%d', localtime()))
normal G
normal o
normal o
endf
autocmd bufnewfile *.py call HeaderPython()
